# AQI Monitoring & Community Event Management System

## 📌 Description
This is a Django-based web application that displays real-time Air Quality Index (AQI) and allows users to join environmental events.

## 🚀 Features
- User Registration & Login
- Event Management System
- Join Event Functionality
- My Events Dashboard
- AQI Monitoring using API
- PM2.5 and PM10 Data Display

## 🛠 Tech Stack
- Python (Django)
- SQLite / MySQL
- HTML, CSS
- OpenWeather / WAQI API
